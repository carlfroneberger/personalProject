This is a website about me, Carl Froneberger. I will be updating it periodically with new information. It is a part of my sophomore project. It was made using HTML, CSS, JS, and soon, PHP. There is little actual information on the site, it is only a coding playground right now. I am hosting it on an Ubuntu 12.10 32 bit server. It has 512 MB of RAM and 30 GB of storage. It should be a lot faster than a Raspberry Pi connected to my cruddy internet, so let's see if it pays off. I hope to add a usable PHP comment system as soon as I have time. I also hope to sort the links out as soon as is possible. You will encounter many broken links, stylesheets that haven't loaded, and scripts that haven't loaded on this site. I have recently begun hoating my own website. As stated in the about page, I am using a DigitalOcean cloudserver. So far, I have found it far preferable to any other option. I do not have enough money to host a proper web server at my home. Additionally, its stability would be spotty at best. I have found that the best way to edit my files is to have a github repository in my documents. Then, I add the files that I have changed, commit them, and push. Sometimes I use the github Sublime Text plugin and other times the command line. Then, I SSH into my web server and pull from github.
