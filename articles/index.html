My first website

This is a website about me, Carl Froneberger. I will be updating it periodically with new information. It is a part of my sophomore project. It was made using HTML, CSS, JS, and soon, PHP. There is little actual information on the site, it is only a coding playground right now. I am hosting it on an Ubuntu 12.10 32 bit server. It has 512 MB of RAM and 30 GB of storage. It should be a lot faster than a Raspberry Pi connected to my cruddy internet, so let's see if it pays off. I hope to add a usable PHP comment system as soon as I have time. I also hope to sort the links out as soon as is possible. You will encounter many broken links, stylesheets that haven't loaded, and scripts that haven't loaded on this site.
